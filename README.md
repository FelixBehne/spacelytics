## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm or yarn

### Installation

1. Clone the repo

```sh
git clone https://github.com/FelixBehne/spacelytics
```

2. Install NPM packages

```sh
yarn
```

4. Start the development server

```sh
yarn dev
```
